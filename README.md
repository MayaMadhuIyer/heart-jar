# 💜 Heart Jar

Heart Jar is a mood-aware journaling app that helps users reflect on their emotions. Users can log moods via quick inputs or journal entries, and the system analyzes them to deliver personalized, supportive quotes inspired by spiritual and philosophical teachings, promoting mindfulness and emotional well-being.

## 🚀 Getting Started

To run this on your computer:

1. Download the ZIP file  
2. Unzip the folder  
3. Open a terminal and navigate to the backend folder, then run:  
   npm start  
4. Open another terminal, navigate to the frontend folder, then run:  
   npx serve .  

## 🌱 Features

- Mood tracking through quick inputs and journaling  
- Intelligent mood analysis  
- Personalized quote recommendations  
- Clean and calming user experience
